## Ready-to-paste Partner Center text

### Product summary

Interactive calendar heatmap for Power BI with daily value coloring, month cards, cross-filtering, and customizable styling.

### Full description

Calendar Heatmap is a custom Power BI visual designed to display daily values in a familiar month-by-month calendar layout.

It helps report authors spot activity patterns, seasonality, gaps, and peaks at the day level while keeping the report experience intuitive for business users.

Key capabilities:

- Month-by-month calendar heatmap layout
- Day-level and month-level cross-filtering
- Editable header title and subtitle
- Dynamic header text fields
- Power BI `fx` support for day color and month card color
- Responsive layout for different report sizes
- Modern card styling for dashboard presentation

Typical use cases:

- sales by day
- attendance tracking
- ticket volume monitoring
- production output analysis
- activity and engagement reporting

## Partner Center flow

1. Create a new `Power BI visual` offer in Partner Center.
2. Fill `Offer setup`.
3. Enable `Request Power BI certification`.
4. Fill `Properties` with legal/support info.
5. Fill `Offer listing` with description, icon, and screenshots.
6. Upload the `.pbiviz` and sample `.pbix` on `Technical configuration`.
7. In certification notes, include the source repo URL and access details.
8. Submit for review.
