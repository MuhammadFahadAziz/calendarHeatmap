# Privacy Policy

Last updated: 2026-05-09

## Calendar Heatmap for Power BI

Calendar Heatmap is a Power BI custom visual that renders report data inside the Power BI environment.

## Data handling

This visual does not send report data to external services.

This visual does not use outbound web requests to transmit report data, user data, or customer data to a remote server.

This visual does not require external web access privileges for its intended functionality.

## Data storage

Calendar Heatmap does not operate a separate hosted backend service for collecting or storing report data.

Any data displayed by the visual is provided by the Power BI report in which the visual is used.

## Personal data

The visual is not designed to independently collect personal data from end users outside the Power BI report context.

If a report author includes personal or sensitive data in a Power BI dataset, that data is governed by the report owner's and tenant administrator's own data governance and privacy practices.
